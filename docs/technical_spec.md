# Technical Specification

## 1. Project Overview

This project is a multi-tenant SaaS platform designed to allow multiple organizations to manage users, projects, and tasks within a single application while ensuring strict tenant data isolation, role-based access control, and subscription plan enforcement. The system follows a monorepo structure with separate backend and frontend services orchestrated using Docker Compose.
